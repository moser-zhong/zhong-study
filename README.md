# 小学五年级英文单词测验

一个专为小学五年级学生设计的英文单词测验工具，具有活泼的界面和完整的题库管理功能。

## 功能特点

### 🎯 测验功能
- 适合小学五年级水平的英文单词测验
- 活泼、色彩丰富的界面设计
- 实时反馈和分数统计
- 进度条显示
- 根据成绩显示不同的鼓励信息

### 📚 题库管理
- 单个单词添加功能
- 批量单词导入功能
- 单词列表查看
- 单词删除功能
- 题库统计信息
- 数据持久化存储

## 技术特点

- 使用HTML5语义化标签
- 响应式设计，支持手机和电脑
- CSS样式与HTML分离
- 完整的JavaScript交互逻辑
- 使用localStorage实现数据持久化
- 不使用内联样式和过时标签

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始测验"进行单词测验
3. 点击"题库管理"添加和管理单词

## 批量导入格式

在题库管理页面，你可以使用以下格式批量添加单词：

```
英文单词,中文释义,干扰选项1,干扰选项2,干扰选项3
apple,苹果,香蕉,橙子,葡萄
book,书,笔,纸,橡皮
cat,猫,狗,鸟,鱼
```
